## Get Started with PipeOps

PipeOps is a Platform as a Service (PaaS) that simplifies the process of deploying and managing applications in the cloud. With PipeOps, you can easily deploy your applications to any cloud provider, including AWS, GCP, and DigitalOcean, with just a few clicks. PipeOps handles all the underlying infrastructure, including building and autoscaling your application, so you can focus on developing and improving your product. Whether you're a developer, operations engineer, or just starting out in the cloud, PipeOps makes it easy to get your applications up and running quickly and efficiently.

## Sign Up

To get started with PipeOps, simply visit the PipeOps [website](https://console.pipeops.io/auth/signup) and sign up for an account. You have the option to authenticate using your Git provider or by using an email address. If you choose email authentication, be sure to confirm your email address to complete the registration process.

## Connect to a Git provider

Connecting to a [Git ](https://console.pipeops.io/dashboard/connect-repo)provider is a crucial step in using PipeOps. By linking your Git provider, PipeOps can automatically create a new deployment every time you push code updates. This makes it easy to continuously deploy your applications and ensure that you have the latest version running in the cloud. For more information on using PipeOps with your Git provider, check out our documentation on the topic.

[PipeOps for GitHub](https://docs.pipeops.io/docs/github)  
[PipeOps for GitLab](https://docs.pipeops.io/docs/gitlab)  
[PipeOps for BitBucket](https://docs.pipeops.io/docs/bitbucket)
