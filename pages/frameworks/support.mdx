## PipeOps Deployment Support
The table below contains a comprehensive list of Pipeops-compatible frameworks, including their compatible build methods for deployment and corresponding run commands.

| Frameworks    | Build Method                        | Run Command                           |
| :------------ | :--------------------------------- | ------------------------------------ |
| Django        | Paketo, Heroku, Dockerfile          | `python manage.py runserver`          |
| Ruby on Rails | Dockerfile, Heroku, Paketo          | `bin/rails server`                    |
| Flask         | Paketo, Heroku, Dockerfile          | `python {{the file}}`                 |
| Express       | Dockerfile, Heroku, Paketo, Node    | `node {{the file}}`                   |
| React         | Paketo, Heroku, Dockerfile, Node    | `npm dev`, `yarn dev`, `pnpm dev`     |
| Angular       | Dockerfile, Heroku, Paketo, Node    | `ng serve`                            |
| Laravel       | Paketo, Heroku, Dockerfile          | `php artisan serve`                   |
| Spring        | Dockerfile, Heroku, Paketo          | `java -jar target/{{project-name}}.jar`|
| ASP.NET       | Paketo, Heroku, Dockerfile          | `dotnet watch`                        |
| Vue.js        | Dockerfile, Heroku, Paketo, Node    | `npm dev`, `yarn dev`, `pnpm dev`     |
| FastAPI       | Paketo, Heroku, Dockerfile          | `python {{the file}}`                 |
| Gin           | Dockerfile, Heroku, Paketo          | `go run {{the file}}`                 |
| Fiber         | Dockerfile, Heroku, Paketo          | `go run {{the file}}`                 |
| Next.js       | Paketo, Heroku, Dockerfile, Node    | `npm dev`, `yarn dev`, `pnpm dev`     |
| Nuxt.js       | Dockerfile, Heroku, Paketo, Node    | `npm dev`, `yarn dev`, `pnpm dev`     |
| Nest.js          | Paketo, Heroku, Dockerfile, Node    | `npm run start`                       |
| Solid.js      | Dockerfile, Heroku, Paketo, Node    | `npm run dev`                         |
